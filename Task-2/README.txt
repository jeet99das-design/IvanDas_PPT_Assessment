# Task 2 – Disk Partitioning (MBR & GPT)

# Objective
To understand and implement disk partitioning using MBR in Ubuntu and GPT in Windows.

# Part A – Ubuntu (MBR)

1. Identified new disk using 'lsblk'.
2. Created partition using 'fdisk'(MBR partition style).
3. Formatted partition using 'mkfs.ext4'.
4. Mounted partition to '/mnt/myvol'.

# Part B – Windows (GPT)

1. Opened Disk Management.
2. Initialized disk using GPT partition style.
3. Created new simple volume.
4. Formatted disk as NTFS.
5. Assigned drive letter.
