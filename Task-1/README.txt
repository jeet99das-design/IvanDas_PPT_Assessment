# Task 1 – Static Web Hosting in EC2 (Ubuntu + Nginx)

# Objective
To deploy a static website on an Ubuntu EC2 instance using Nginx.

# Services Used
- Amazon EC2
- Ubuntu Server
- Nginx Web Server

# Steps Followed
1. Launched Ubuntu EC2 instance.
2. Configured Security Group to allow HTTP (80) and SSH (22).
3. Installed and started Nginx.
4. Downloaded and extracted static website template (Tooplate).
5. Copied website files to '/var/www/html/'.
6. Verified website access using Public IP.
