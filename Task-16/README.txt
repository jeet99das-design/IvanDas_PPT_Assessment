==============================
Task 16: RDS Database + SQL Workbench Connection & Queries
==============================

Objective:
To create a managed relational database using Amazon RDS and connect using SQL Workbench.

Services Used:
- Amazon RDS
- MySQL Engine
- SQL Workbench

Implementation Steps:
1. Created RDS instance using MySQL engine.
2. Configured security group to allow MySQL port (3306).
3. Created database and master credentials.
4. Connected to RDS using SQL Workbench.
5. Executed sample SQL queries (CREATE TABLE, INSERT, SELECT).

Outcome:
Successfully connected to RDS and executed SQL queries.
