==============================
Task 18: Azure Virtual Machine
==============================

Objective:
To launch and configure a basic Virtual Machine in Microsoft Azure.

Services Used:
- Azure Virtual Machine
- Azure Resource Group
- Network Security Group

Implementation Steps:
1. Created a resource group.
2. Launched a virtual machine (Windows/Linux).
3. Configured NSG rules for RDP/SSH.
4. Connected to VM using RDP/SSH client.

Outcome:
Azure VM successfully deployed and accessed remotely.
