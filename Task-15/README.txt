==============================
Task 15: CloudFront Distribution for S3 Static Website
==============================

Objective:
To improve website performance using CDN.

Services Used:
- Amazon S3
- Amazon CloudFront

Implementation Steps:
1. Created S3 bucket and enabled static website hosting.
2. Uploaded website files (HTML/CSS).
3. Created CloudFront distribution.
4. Linked CloudFront origin to S3 bucket.
5. Accessed website using CloudFront domain name.

Outcome:
Website successfully delivered via CloudFront with improved performance and global distribution.
