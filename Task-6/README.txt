# Task 6 – S3 Replication and Lifecycle Policy

# Objective
To configure cross-region replication and lifecycle policy in Amazon S3.

# Services Used
- Amazon S3

# Replication Steps Performed:
1. Created source and destination S3 buckets.
2. Enabled versioning on both buckets.
3. Created replication rule in source bucket.
4. Uploaded test file in source bucket.
5. Verified automatic replication in destination bucket.

# Key Learning

- Versioning is mandatory for replication.
- Replication helps in disaster recovery.
- Lifecycle policies optimize storage cost.
