# Task 8 – Create IAM Users and Groups

# Objective
To create IAM users and manage permissions using IAM groups.

# Service Used
- AWS Identity and Access Management (IAM)

# Implementation Steps

1. Created IAM user group.
2. Attached AmazonS3FullAccess policy to the group.
3. Created IAM user.
4. Added user to the group.
5. Verified user access permissions.

# Key Learning

IAM groups simplify permission management by assigning policies to multiple users at once.
