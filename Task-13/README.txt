==============================
Task 13: Start/Stop EC2 using Lambda + API Gateway
==============================

Objective:
To automate EC2 instance start and stop operations using API trigger.

Services Used:
- AWS Lambda
- API Gateway
- Amazon EC2
- IAM Roles

Implementation Steps:
1. Created Lambda function with EC2 start/stop permissions.
2. Wrote Python code to start and stop EC2 instance.
3. Created REST API using API Gateway.
4. Integrated API Gateway with Lambda function.
5. Deployed API and tested using endpoint URL.

Outcome:
Successfully controlled EC2 instance state using API endpoint.
